Learning Python web

Python Web的学习路线

第一步：python入门

——《python简明教程》，适合小白入门；

第二步：python进阶

——《python学习手册》，如面向对象的这部分对于小白理解相对容易。《python cookbook》——函数，装饰器以及网络编程这三部分讲得真的不错。

第三步：学习web开发，涉及到基于python的web 框架，有下面三个：

  • flask：推荐《flask web 开发》这本书，这本书简直是良心，讲的很细。还有就是外文：《building web applications with flask》《flask Framework cookbook》

  • django：django是大而全，开箱即用。django学习可以看得书：

    • tango with django
      • django by example 在学习框架的过程中，会不断体会到前端的知识也是要补回来，可以w3c school看一下。

第四步：数据库

在python中内置了sqlite， sql命令有相同之处，路线是sqlite， mysql，mongodb，有个逐渐过度的过程。

第五步：了解http的具体工作流程

第六步：工具集

+ sublime(需要配置，主要用于python编程，神器谁用谁知道)， 
+ vim， 
+ firebug, 
+ firefox, 
+ linux,
+ github（代码大宝库，各种代码）.
